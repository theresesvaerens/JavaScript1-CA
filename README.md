This project is my JavaScript 1 Course Assignment, where the goal was to create an interactive online store using data from an API.
I chose to continue working on my existing Square Eyes project and used the Square Eyes API to dynamically load product data.

The main focus of the assignment was to demonstrate practical JavaScript skills such as fetching data, handling errors, manipulating the DOM, and building interactive features like a shopping basket.

# Project Goal
The purpose of this assignment was to:
- Fetch and display products from an API
- Build an interactive online store
- Allow users to view, filter and interact with content
- Enable users to add/remove items from a shopping basket
- Provide a simple checkout flow
- Use clean, structured and readable JavaScript
- Avoid hardcoded product data
- Handle loading states and API errors

# User Stories
This project meets the required user stories:
- View product list: Users can view all items on the homepage
- Filter products: Users can filter movies by genre
- View single product: Users can open a detailed product page
- Add to basket: Users can add products to a shopping basket
- Remove from basket: Users can remove items at any time
- View checkout summary: Users can see their selected items in the checkout
- See order confirmation: Users receive a confirmation message after checkout
